Project: `Jekyll`
===============
"Project: `Jekyll`" is a datapack for `Minecraft: Java Edition 1.21.10`.

The end-goal is to add many monsters to the game, along with drops that allow the player to gain their abilities.

Current Monsters:
-----------------
- Vampire
	- Strength
	- Speed
	- Flight
	- Lifesteal

Planned Monsters:
-----------------
- Werewolf
